Introducción.

HTML es el lenguaje que se usa para hacer estructuras de front-end de páginas web.

HTML son las siglas de "HyperText Markup Language" (En español "Lenguaje de marcas de hipertexto"). A diferencia de los lenguajes de programación, un lenguaje de marcas como lo es HTML utiliza etiquetas para identificar y diferenciar partes del contenido.

Diseño web front-end moderno:

  • HTML:
    Estructura.
  • CSS:
    Presentación.
  • JavaScript:
    Comportamiento front-end.

Estructura de un documento.

Todo el documento se escribe entre las etiquetas <html> y </html>, el documento a su vez se divide en la cabeza y el cuerpo, delimitadas por las etiquetas <head> y <body>.

Ejemplo:

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
    </body>
</html>

Comentarios.

Los comentarios sirven para hacer anotaciones en el código que no se verán en el resultado final. Lo normal es que se utilicen para explicar partes de códigos que sean poco entendibles o aclaraciones más simples.

Los comentarios en HTML se escriben de la siguiente forma:

Sintaxis:

<!-- Esto es un comentario -->
