Creación de una página web offline.

Para crear una página web offline que se ejecutará en el navegador directamente debes escribir un archivo de texto con las siguientes etiquetas escritas dentro:

Código básico de una página web:

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
    Texto de mi página web...
    </body>
</html>
Copia el código y ejecútalo para ver cómo funciona!

Ejecutar una página web offline.

  1. Copia el código del ejemplo de la sección anterior y pégalo en un bloc de notas.
  2. A continuación, guárdalo con la extensión '.html'.
  3. Ejecuta el archivo con tu navegador preferido.
