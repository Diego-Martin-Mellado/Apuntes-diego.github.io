Etiquetas - HTML 5.

<html>



Etiquetas main entre las que se ubica todo código en HTML.

<head>



Etiquetas entre las que se encuentran los metadatos de tu página web.

<body>



Etiquetas entre las que se encuentra la parte visible de una página web.

<a>



Etiqueta usada para crear enlaces tanto a otras páginas web como a una parte de la misma página.

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>



Los 6 tipos de títulos de encabezados de HTML 5

<img>



Etiqueta de HTML que permite introducir imágenes en tu página de HTML.

<p>



Etiqueta para crear un párrafo en tu página HTML.

<title>



Etiqueta para poner título a tu página web.
