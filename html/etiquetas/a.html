Etiqueta <a>.

Las etiquetas <a> y a> es la etiqueta de HTML en la que se especificará una imágen y sus parámetros para incluirla en tu página web, ésta etiqueta sólo puede situarse en el apartado <body>.

Ejemplo:

<a href = " valor " >

Valor.

El valor que se escribe entre las comillas del atributo href puede ser de 3 índoles, a saber:

  • Link a una página externa:
    En este caso, el contenido del atributo href puede ser una URI (URL + URN) (Protocolo + :// + Ruta + ? + paramteros + # + hash)

  • Link a un lugar dentro de la página:
    En este caso, el contenido del atributo href debe ser el valor de algún atributo id perteneciente a algún elemento dentro de la misma página. Siempre ha de estar precedido de el caracter (# ).

  • Link a envío de un correo:
    En este caso, el contenido del atributo href debe estar formado por la palabra mailto: seguido de la dirección de correo a quien se quiera enviar el mensaje. Opcionalmente se pueden añadir el asunto (subject), cuerpo del correo (body), correos con copia (cc:) y con copia oculta (bcc:).
    Todos los espacios en los atributos del correo deben ser sustituidos por el caracter %20 y en caso de que se quiera añadir más de un valor a un atributo se deberán separar por puntos y coma (;)

Ejemplos mas concretos:

<a href="https://www.codecademy.com/">
<ahref="#cabecera">
<ahref="mailto:nombre@organizacion.com;nombre2@organizacion.com?cc=nombreconcopia@organizacion.com &bcc =concopiaoculta@organizacion.com &subject =Mensaje &body =Este%20es%20el%20cuerpo%20del%20correo">
