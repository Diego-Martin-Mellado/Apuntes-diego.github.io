Etiqueta <title>.

Las etiquetas <title> y </title> son las etiquetas de HTML de apertura y cierre entre las que se encontrará el título de tu página web, ésta etiqueta se encuentra en la etiqueta <head>.

Ejemplo:

<title>Título de mi página web</title>
La etiqueta title sólo puede ser escrita una vez en cada página web, y no tiene atributos.
