Etiqueta <html>.

Las etiquetas <html > y </ html > son las etiquetas de HTML de apertura y cierre entre las que se encontrará todo el contenido de tu página web.

Entre éstas etiquetas se encuentran 2 secciones que se explicarán en las próximas entradas, son:
  • <head > y </ head >
  • <body > y </ body >
