Librería estándar - JavaScript

JavaScript Browser Object Model



Dado que los navegadores modernos han implementado (casi) los mismos métodos y propiedades para la interactividad de JavaScript, a menudo se lo conoce como métodos y propiedades de la lista de materiales.

JavaScript Document Object Model



Cuando se carga una página web, el navegador crea un modelo de objetos de documento de la página. El modelo HTML DOM se construye como un árbol de objetos.
