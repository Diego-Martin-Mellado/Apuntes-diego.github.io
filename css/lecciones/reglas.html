Reglas - CSS.

Una regla CSS es una agrupación de una o más propiedades de CSS que se aplicarán a uno o más elementos HTML 5 de destino.

Una regla CSS consiste en un selector CSS y un conjunto de propiedades CSS. El selector CSS determina a qué elementos de HTML 5 apuntar con la regla CSS. Las propiedades CSS especifican el estilo de los elementos HTML 5 objetivo.

Partes de una regla:

      Con una propiedad:
selector { propiedad: valor; }
      Con varias propiedades:
selector {
    propiedad: valor;
    propiedad: valor;
    propiedad: valor;
}
Las propiedades y sus posibles valores se pueden ver en el apartado de librería estándar.
