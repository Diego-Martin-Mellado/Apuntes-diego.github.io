Background-color - CSS.

La propiedad background-color en CSS indica el color que tendrá el el fondo de la o las etiquetas indicadas por el selector.

Los colores pueden estar indicados de diversas formas, a saber:

  • Valor en hexadecimal RBG.
  • Nombre.
  • Funcion en rgb()
  • Funcion en rgba()
  • Funcion en hsl()
  • Funcion en hsla()

Ejemplos de background-color - CSS.

Ejemplo de propiedad background-color:

.rojo {
  background-color: red;
}
.rojo {
  background-color: #FF0000;
}
