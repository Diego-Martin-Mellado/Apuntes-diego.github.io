Arrow - Bulma.

La función arrow () crea una flecha apuntando hacia abajo, recibe un parámetro que indica el color de la flecha.

Se debe especificar en un archivo .css propio el color de la flecha.
Uso:
CSS:
.mi-flecha {
    @include arrow (#FF00FF);
}
HTML:
<span class="mi-flecha"> </span>
