Introducción - Bootstrap.

Bootstrap es una librería de CSS multiplataforma orientada a la simplificación del diseño web.

Quizás es la más famosa por su simplicidad de uso y la gran cantidad de clases de CSS y posibilidades predefindidas.

Boostrap favorece enormemente la creación de páginas web responsive.

Una página web responsive es la que se adapta a la resolución del dispositivo en la que se visualiza, de manera que se puede ver diferente en móvil, tablet, ordenador o televisión aun teniendo un solo archivo HTML
Hacer una página con la ayuda de Bootstrap es muy cómodo sobre todo en lo relacionado con el manejo del modelo de cajas.
