Módulo urls - Python

El módulo django.urls.

Funciones de django.urls - Python

Función include



La función django.urls.include () cambia una ruta de importación completa de Python a otro módulo URLconf que debería estar "incluido" en este lugar.

Función path



La función django.urls.path () crea y retorna un elemento para incluirlo en el array urlpatterns.
