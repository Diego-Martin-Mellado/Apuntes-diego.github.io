Django - Python

Django es un framework de alto nivel dedicado al desarrollo web con Python. Django se encarga de gran parte de las complicaciones del desarrollo web, por lo que con poco código puedes obtener grandes resultados.

Django es gratis y de código abierto, tiene una comunidad muy activa, una buena documentación y muchas opciones de soporte.

Se basa en el modelo de programación MVT (Model View Template) (Modelo Vista Plantilla en castellano).

Modelo Vista Template (MVT):

Éste modelo de programación se basa en recibir la petición del cliente o visitante web, se pasará a la vista, en la vista recuperaremos los datos del modelo correspondiente, y finalmente la renderizaremos el Template pero esta vez integrando los datos dinámicos recuperados del modelo, antes de enviar el HTML final al navegador.
La versión de Python mínima requerida para utilizar Django es 3.6.

Instalación:

~$: pip install django
Dependencias:
  • pytz
  • sqlparse >= 0.2.2
  • asgiref < 4 & asgiref >= 3.3.2
