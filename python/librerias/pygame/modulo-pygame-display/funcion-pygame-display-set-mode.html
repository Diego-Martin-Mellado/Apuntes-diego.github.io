Función pygame.display.set_mode - Python

La función pygame.display.set_mode () inicializa una ventana o pantalla para programar sobre ella.

De sus parámetros, el único obligatorio es el parámetro size, que indica el tamaño de la ventana. Que en realidad, si no se añade, funciona perfectamente, pero la ventana creada medirá 0x0.

Sintaxis:

mi_ventana = pygame.display.set_mode (size = (0, 0), flags = 0, depth = 0, display = 0)
Referencia:

  • Size:
    Es una tupla (tuple), con dos números que indican el ancho y el alto de la ventana.

  • Flags:
    Es una colección de argumentos opcionales.

  • Depth:
    Es el número de bits para usar por cada color.

  • Display:
    No he encontrado información sobre éste parámetro, pero se que existe, si encuentras información acerca de qué es, mándame un correo.

Retorno:

Surface

Uso de pygame.display.set_mode - Python

Uso de pygame.display.set_mode ():

import pygame

pygame.init ()
pantalla = pygame.display.set_mode ((500, 500)) // Fijo el ancho y el alto de la ventana a 500

salir = False

while not (salir):
    for event in pygame.event.get ():
        if event.type == pygame.QUIT:
            salir = True
pygame.quit ()
