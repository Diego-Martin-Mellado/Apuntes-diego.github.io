Función pygame.display.set_icon - Python

La función pygame.display.set_icon () establece un icono en la ventana inicializada por PyGame.

Su único parámetro es la propia imagen ya cargada como superficie de pygame.

Sintaxis:

pygame.display.set_icon (source)
Referencia:

  • Source:
    Es una superficie (Surface), es la imagen previamente cargada en una superficie de PyGame.


Retorno:

None (NoneType)

Uso de pygame.display.set_icon - Python

Uso de pygame.display.set_icon ():

import pygame

pygame.init ()
pantalla = pygame.display.set_mode ((500, 500)) // Fijo el ancho y el alto de la ventana a 500
pantalla = pygame.display.set_icon (pygame.image.load ('ruta/a/la/imagen.png')) // Fija la imgaen de icono

salir = False

while not (salir):
    for event in pygame.event.get ():
        if event.type == pygame.QUIT:
            salir = True
pygame.quit ()
