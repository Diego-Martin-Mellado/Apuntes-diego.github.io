PyGame - Python

Pygame es un conjunto de módulos de Python diseñados para escribir videojuegos descargada millones de veces. Pygame agrega funcionalidad además de la excelente biblioteca SDL. Esto le permite crear juegos y programas multimedia con todas las funciones en el lenguaje Python.

Pygame es portable, se ejecuta en casi todas las plataformas y sistemas operativos.

Pygame está lanzada bajo la licencia LGPL, puede crear juegos de código abierto, freeware, shareware y comerciales con él. Consulte la licencia para obtener todos los detalles.

Éste módulo no tiene una versión de Python mínima requerida, aunque funciona mejor en versiones recientes.
Instalación:

~$: pip install pygame
Ésta librería no tiene dependencias.

Modulos de PyGame - Python

Modulo pygame.cdrom



Es el módulo que controla los drivers de CDs y DVDs de un ordenador.

Modulo pygame.cursors



Control sobre la apariencia del cursor del usuario mientras se ejecuta un programa.

Modulo pygame.display



Representa el control en la ventana del juego o la pantalla en caso de que esté maximizado.

Modulo pygame.draw



Submódulo para crear formas básicas con PyGame.

Modulo pygame.event



Manejador de eventos básico de PyGame.

Modulo pygame.font



Da el control sobre las fuentes de todos los textos o "label" de un juego.

Modulo pygame.joystick



Módulo que proviene de funciones para crear videojuegos juegos jugables con joystick.

Modulo pygame.key



Módulo de PyGame para controlar el teclado.

Modulo pygame.mouse



Módulo de PyGame para controlar el ratón.

Modulo pygame.sndarray



Mezcla de NumPy y objetos Sound para controlar el audio de manera más simple.

Modulo pygame.surfarray



Manera de PyGame para controlar imágenes y superficies a través de arrays (list).

Modulo pygame.time



Módulo de PyGame para controlar de manera simple el tiempo y la cronología de eventos en un juego.

Modulo pygame.transform



Provee de funciones para el manejo de imágenes y figuras dentro de un videojuego de PyGame.
