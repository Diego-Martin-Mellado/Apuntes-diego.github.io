Covid - Python

Se trata de una librería de Python dedicada a recoger información acerca de el Covid-19, una pandemia que afectó al mundo en el año 2020.

La librería covid por defecto recoge datos de la Universidad de Johns Hopkins, que es una universidad privada de Baltimore, Maryland, Estados Unidos, aunque también se puede indicar que los recoja de la página https://www.worldometers.info/ .

La versión de Python mínima requerida para utilizar covid es 3.6 .

Instalación:

~$: pip install covid
Dependencias:

  • pydantic
  • requests
