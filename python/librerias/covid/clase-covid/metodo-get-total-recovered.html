Covid - Python

El método get_total_recovered () se aplica al objeto covid.Covid () de la librería covid.

El método covid.Covid ().get_total_recovered () devuelve el número de personas que se han recuperado de la enfermedad Covid-19 en el mundo.

Llamada:

cov_obj.get_total_recovered ()
Retorno:

int

Ejemplos de get_total_recovered - Python

Uso de Covid.get_total_recovered ():

// Uso estándar
import covid

cov_info = covid.Covid ()

print (cov_info.get_total_recovered ()) // Imprime los casos recuperados de coronavirus (Covid-19) actuales
