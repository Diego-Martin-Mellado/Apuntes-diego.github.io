Covid - Python

El método get_total_deaths () se aplica al objeto covid.Covid () de la librería covid.

El método covid.Covid ().get_total_deaths () devuelve el número de personas del mundo que han muerto por la enfermedad Covid-19 en el mundo.

Llamada:

cov_obj.get_total_deaths ()
Retorno:

int

Ejemplos de get_total_deaths - Python

Uso de Covid.get_total_deaths ():

// Uso estándar
import covid

cov_info = covid.Covid ()

print (cov_info.get_total_deaths ()) // Imprime las muertes por coronavirus (Covid-19) actuales
