Librería estándar - Python

Builtins



Funciones, clases, metodos, sintaxis y datos de la librería estándar de Python lo más actualizado posible.

Modulo os



El módulo estándar de Python dedicado la interactuación entre el sistema operativo del cliente y el programa que se ejecuta.

Modulo sys



El módulo sys da acceso al programador a todas las variables e información del intérprete y el sistema en el que se está ejecutando un programa.
