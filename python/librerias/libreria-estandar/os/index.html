Módulo os - Python

El módulo os da acceso al programador directamente a todas las variables e información del intérprete y el sistema en el que se está ejecutando un programa.

Entre sus funciones incluye:
  • Acceso a argumentos.
  • Finalización del programa en cualquier momento.
  • Información de la ejecución del programa.
  • ...

Módulo os - Python

Funciones de os.




Que puedes utilizar en tus programas.

Funcion os.mkdir ()



La función os.mkdir () sirve para crear una carpeta en el ordenador del cliente.

Funcion os.system ()



Función para ejecutar comandos en la consola del sistema. Igual que la función system () de C++.
