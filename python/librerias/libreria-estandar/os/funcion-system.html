Función system - Python

La función os.system () ejecuta un comando en la consola de comandos del sistema operativo del cliente.

Ésta función se corresponde con la función de C/C++ system () de la librería <stdlib.h>.
Sintaxis:

os.system (str comando)
Referencia:

  • Comando:
    Es un string (str) que se va a ejecutar directamente en la consola de comandos del cliente.

Retorno:

None

Uso de system

Ejemplos de uso de system ():

// Limpia la consola de Linux
os.system ('clear')

// Limpia la consola de Windows
os.system ('cls')
