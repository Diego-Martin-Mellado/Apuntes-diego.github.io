Función mkdir - Python

La función os.mkdir () sirve para crear una carpeta en el ordenador del cliente.

os.mkdir () requiere de mínimo 1 parámetro, éste deberá ser un str con el nombre de la carpeta que se quiere crear.

Sintaxis:

os.mkdir (str path, int mode = 0o777, file dir_fd = None)
Referencia:

  • path:
    Fija el nombre del directorio.
  • mode:
    Indica los permisos que se dan al directorio.
  • dir_fd:
    Indica el archivo que contiene la descripción del directorio.

Retorno:

None

Uso de mkdir

Ejemplos de uso de mkdir ():

// Crea una carpeta en el directorio activo con el nombre 'carpeta1'
os.mkdir ('carpeta1')
