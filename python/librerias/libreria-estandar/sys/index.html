Módulo sys - Python

El módulo sys da acceso al programador directamente a todas las variables e información del intérprete y el sistema en el que se está ejecutando un programa.

Entre sus funciones incluye:
  • Acceso a argumentos.
  • Finalización del programa en cualquier momento.
  • Información de la ejecución del programa.
  • ...

Variable sys.argv



La variable sys.argv es una variable de tipo list que contiene los argumentos pasados a la ejecución de un programa en Python.

Funcion sys.exit ()



La función sys.exit () sirve para detener el programa de python que se está ejecutando.
