Función sys.exit() - Python

La función sys.exit () se encuentra en el módulo sys de la librería estándar de Python.

La función sys.exit () sirve para detener el programa de python que se está ejecutando, desde cualquier ámbito, vease, desde cualquier lugar del programa, ya sea en el ámbito global, en cualquier condición, bucles e incluso funciones o métodos en clases.

El argumento es un número entero entre el 0 y el 127. Aunque si dejas vacío el parámetro, se sobreentiende 0 o salida exitosa. Si se pasa None, tambien se sobreentienede como 0.

En caso de que se pase un dato de tipo str, se pasará como mensaje de error al sistema.

Sintaxis:

sys.exit()
Sintaxis:

sys.exit(int exito)
Referencia:

  • Éxito:
    El código de retorno que debe dar el programa al finalizar.
Sintaxis:

sys.exit(str mensaje)
Referencia:

  • Mensaje:
    Mensaje que se dará al sistema escrito en el canal sys.stderr.
Retorno:

Ninguno

Uso de sys.exit() - Python

// Función para parar el programa
def parar (arg):
  // [...]
  sys.exit (arg)
