Método lower () - Python

El método lower () es un método del tipo de dato str de Python.

El método lower () sirve para convertir todas las letras de un str (string) a minúsculas.

Sintaxis:

str.lower ()
Retorno:

str

Ejemplos de lower - Python

Ejemplos de uso del método str.lower ().
nombre = input ("Escribe un nombre: ")
if nombre.lower () == 'diego':
  print ('Igual que yo!')
// Admite entradas en cualquier orden de mayusculas y minusulas, es decir: 'Diego', 'DiEgO', 'dIeGo' ...
