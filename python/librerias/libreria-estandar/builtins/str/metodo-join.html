Método join () - Python

El método join () es un método del tipo de dato str de Python.

El método join () sirve para concatenar los objetos de un iterable en un string.

El separador utilizado para unir los objetos de dicho
iterable
será el contenido del str al que se le aplica el método.
Los valores str que se utilizan para representar los objetos del iterable en el str resultante se obtienen del método ()
Sintaxis:

str.join (iterable object)
Retorno:

str

Ejemplos de join - Python

Ejemplos de uso del método str.join ().
nombre = ["Sandra", "del", "Pozo"]
print (' '.join (nombre)) // Imprime 'Sandra del Pozo'
