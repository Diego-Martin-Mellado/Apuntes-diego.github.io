Método capitalize () - Python

El método capitalize () es un método del tipo de dato str de Python.

El método capitalize () sirve para convertir la primera letra de un str en mayúscula.

Sintaxis:

str.capitalize ()
Retorno:

str

Ejemplos de capitalize - Python

Ejemplos de uso del método str.capitalize ().
nombre = 'sandra'
print (nombre.capitalize ()) // Imprime 'Sandra'
