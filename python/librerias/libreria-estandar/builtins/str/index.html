Tipo str - Python

La clase str representa el tipo de dato "string" en el lenguaje de programación Python.

Se escribe entre comillas dobles ("), comillas simples ('), comillas inclinadas (`), en los que un salto de línea en el programa será interpretado como un error de interpretación, o triples comillas dobles ("""), con ésta última modalidad, se pueden incluir saltos de línea que no serán interpretados como errores.

La modalidad de str que se escribe con triples comillas dobles normalmente es usada como comentarios (docstring).
Los objetos creados con la clase str son iterables, es decir, puedes recorrer el objeto con un bucle, ya sea for o while.

Métodos de str - Python

Metodo capitalize ()



El método .capitalize () convierte la primera letra de un str a mayúscula.

Metodo join ()



El método .join () concatena los objetos de un iterable en un string.

Metodo lower ()



El método .lower () sirve para convertir todas las letras de un str (string) a minúsculas.

Metodo upper ()



El método .upper () sirve para convertir todas las letras de un str(string) a mayúsculas.
