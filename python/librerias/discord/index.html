Discord - Python

La versión de Python mínima requerida para utilizar discord.py es 3.5.3 .

Instalación:

~$: pip install discord.py
Dependencias:

  • libffi
  • libnacl
  • python3-dev

Modulos de Discord - Python

Modulo discord.ext.commands



Ofrece una manera de interactuar con Discord a más bajo nivel.

Modulo discord.ext.tasks



Una de las operaciones más comunes que se hacen mientras construyes un Bot de discord es tener un bucle ejecutandose en segundo plano, ésta librería proporciona herramientas para hacerlo más fácilmente.
