Playsound - Python

La función playsound.playsound () reproduce un sonido en el ordenador del cliente que está ejecutando el programa.

Sintaxis:

playsound.playsound (str path)

Referencia:

  • Path:
    Ruta del archivo de audio que se quiere repoducir.

Retorno:

None

Uso de playsound - Python

Uso de la funcion playsound.playsound ():

import playsound

// Reproduce el audio en la ruta 'ruta/a/archivo.mp3'
playsound.playsound ('ruta/a/archivo.mp3')
from playsound import playsound

// Reproduce el audio en la ruta 'ruta/a/archivo.mp3'
playsound ('ruta/a/archivo.mp3')
