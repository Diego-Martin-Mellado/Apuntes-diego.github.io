Playsound - Python

La librería de Python playsound es una librería de Python que contiene una función que sirve para reproducir sonidos en el ordenador del cliente.

La librería playsound es multiplataforma y se puede instalar fácilmente con la herramienta pip, como la mayoría de Python.

La versión de Python mínima requerida para utilizar playsound es 2.3 .

Curiosidad.

Está declarada como la librería de Python más pequeña del mundo.
Instalación:

~$: pip install playsound

Funcion de Playsound - Python

Funcion playsound()



La función playsound.playsound () reproduce un sonido en el ordenador del cliente que está ejecutando el programa.
