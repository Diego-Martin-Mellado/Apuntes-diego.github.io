WatchDog - Python

La librería de Python watchdog es una librería de Python que contiene herramientas para operar con la consola y monitorear eventos que ocurren en el sistema de archivos del sistema.

La librería watchdog es multiplataforma y se puede instalar fácilmente con la herramienta pip , como la mayoría de Python.

La versión de Python mínima requerida para utilizar watchdog es 3.5.3 .

Instalación:

~$: pip install watchdog
Dependencias:

  • Python 2.7+ / Python 3.4+
  • pathtools
  • PyYAML
  • argh

Modulos de WatchDog - Python

Modulo watchdog.events



El módulo watchdog.event se encarga de gestionar el sistema de ficheros y el manejador de eventos de la librería.

Modulo watchdog.observers



Se trata del núcleo principal de la API, contiene una única clase llamada Observer.

Modulo watchdog.utils



Contiene clases y funciones que pueden servir de ayuda a la hora de monitorear archivos y directorios.
