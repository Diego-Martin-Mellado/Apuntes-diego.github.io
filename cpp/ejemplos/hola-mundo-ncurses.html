Hola mundo (<ncurses.h>) - C++.

Este código imprime por consola "Hola mundo!" con la librería <ncurses.h>.

// Programa básico con <ncurses.h>
#include <ncurses.h>

int main (int argc, char *argv[]) {
  initscr (); // Inicializa la ventana
  printw ("Hola mundo!"); // Imprime "Hola mundo" en la posición (0, 0)
  refresh ();
  getch ();
  endwin (); // Cierra la ventana
  return 0;
}
Recuerda compilar con la siguiente línea:

g++ archivo.cpp -o ejecutable -lncurses
