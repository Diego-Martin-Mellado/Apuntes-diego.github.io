<ncurses.h> - C++.

La librería <ncurses.h> es exclusiva de sistemas Linux.
Puedes instalar <ncurses.h> ejecutando el siguiente comando en la consola de Linux:

sudo apt-get install ncurses-dev
Al compilar con g++, debes especificar que estás usando <ncurses.h> añadiendo el parámetro -lncurses de la siguiente manera:

g++ archivo.cpp -o ejecutable -lncurses
