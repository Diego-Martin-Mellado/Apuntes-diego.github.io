Función sizeof.

sizeof () es una función que se encuentra en la librería <stdio.h>.

sizeof () retorna el tamaño en bytes del dato pasado por parámetro.

Sintaxis:

sizeof ( class dato );
Referencia:

  • Dato:
    Es la variable de la cual se va a indicar su tamaño en bytes, puede ser de cualquier tipo, incluyendo clases definidas por el programador.

Retorno:

unsigned int

Ejemplos de uso de sizeof.

Ejemplos de uso de sizeof ():

// Ejemplo
char *dato = "MiString";
printf ("%u\n", sizeof ( dato )); // Imprime la longitud de un char, para imprimir el tamaño de toda la palabra ve aqui.
