Función round.

round () es una función que se encuentra en la librería <math.h>.

round () sirve para redondear un número que es pasado por parámetro.

Sintaxis:

round ( double/long double/float numero );
Referencia:

  • Número:
    Número a redondear.

Retorno:

(double/long double/float) (El mismo tipo de dato que el pasado por parámetro).

Ejemplos de uso de round.

Ejemplos de uso de round ():

double redondeo = round ( 3.14 ); // redondeo vale 3.0
double redondeo2 = round ( 2.71 ); // redondeo2 vale 2.0
