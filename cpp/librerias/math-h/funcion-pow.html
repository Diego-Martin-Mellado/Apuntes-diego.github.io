Función pow.

pow () es una función que se encuentra en la librería <math.h>.

pow () sirve para hacer una potencia en la cual la base es el primer parámetro y el exponente es el segundo.

La función pow () puede devolver los siguientes errores en ciertos casos:

Si la base es negativa y el exponente no es un valor numérico: Error: domain error.
Si tanto la base como el exponente son 0: Error: domain error.
Si la base es 0 y el exponente es negativo: Error: domain error o pole error.
Si el resultado es más grande o más pequeño que el admitido por la variable destino: Error: range error.
Sintaxis:

pow ( class base, class exponente );
Referencia:

  • Base:
    Base de la potencia. Puede ser de tipo int /float /double.

  • Exponente:
    Exponente de la potencia. Puede ser de tipo int /float /double.

pow (base, exponente) = base exponente
Retorno:

float

Ejemplos de uso de pow.

Ejemplos de uso de pow ():

double potencia = pow ( 2, 3 ); // potencia vale 8.0
double potencia2 = pow ( potencia, 3 ); // potencia2 vale 8.0 ^ 3 = 512.0
