Función strlen.

strlen () es una función que se encuentra en la librería <string.h>.

strlen () te indica el número de caracteres que tiene una cadena de caracteres (variable tipo char *).

La función strlen () no cuenta el caracter '\0'.
Sintaxis:

strlen (char *cadena);

Referencia:

  • Cadena:
    Es el string al que se le van a contar los caracteres.

Retorno:

int

Ejemplos de uso de strlen.

Ejemplos de uso de strlen ():

char *cadena = "Ejemplo";

printf ("%i\n", strlen (cadena) );
