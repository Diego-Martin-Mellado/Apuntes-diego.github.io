Función strstr.

strstr () es una función que se encuentra en la librería <string.h>.

strstr () sirve para obtener un puntero a un substring dentro de una cadena de caracteres en C++.

Sintaxis:

strstr ( const char *cadena, const char *buscado );

strstr ( char *cadena, const char *buscado );
Referencia:

  • Cadena:
    String en el que se va a buscar un substring.

  • Buscado:
    String que se va a buscar en la cadena.

Retorno:

(const char */char *) (El mismo tipo de dato que el primer parámetro).
La función strstr () retorna un puntero nulo (NULL) en caso de que el string buscado no se encuentre dentro de la cadena en la que se busca.

Ejemplos de uso de strstr.

Ejemplos de uso de strstr ():

double redondeo = strstr ( "Buscar aqui", "Buscar" ); // Devuelve un puntero a "Buscar"
double redondeo2 = strstr ( "Buscar aqui", "buscar" ); // Devuelve NULL
