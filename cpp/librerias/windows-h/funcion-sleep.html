Función Sleep.

Sleep () es una función que está en la librería <windows.h>

Sleep () hace una pausa en el programa del tiempo indicado en milésimas de segundos, después, continúa con las siguientes instrucciones.

Sintaxis:

Sleep ( int milisegundos );
Referencia:

  • Milisegundos:
    Es el número de milésimas de segundo que el programa va a estar esperando antes de ejecutar la siguiente instrucción.

Retorno:

void

Ejemplos de uso de Sleep.

Ejemplos de uso de Sleep ():

Sleep (1000); // Espera 1 segundo

Sleep (3456); // Espera 3.456 segundos
