Función islower.

islower () es una función que se encuentra en la librería <ctypes.h>.

islower () indica si el parámetro recibido (char) es un caracter en minúscula, éste caracter también puede ser un entero (int) que será pasado a caracter antes de evaluar.

islower () es la función con el efecto contrario a isupper ().
Sintaxis:

islower ( char/int caracter);

Referencia:

  • Caracter:
    Caracter a evaluar si es minúsuculas.

Retorno de islower ().

bool

Ejemplo de uso de islower.

Uso de islower ():

char caracter = 'F';
int caracter2 = 100;

bool esMinusculas1 = islower (caracter); // Retorna false
bool esMinusculas2 = islower (caracter2); // Retorna true (porque es el codigo de la 'd')
