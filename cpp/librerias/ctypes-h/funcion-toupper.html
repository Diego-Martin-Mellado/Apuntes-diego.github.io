Función toupper.

toupper () es una función que se encuentra en la librería <ctypes.h>.

toupper () pasa el parámetro recibido (char *) a mayúsculas.

toupper () es la función con el efecto contrario a tolower ().
Sintaxis:

toupper ( char *cadena );

Referencia:

  • Cadena:
    Cadena de caracteres que se va a pasar a mayúsculas

Retorno de toupper ().

char *

Ejemplo de uso de toupper.

Uso de toupper ():

char *cadena = "CrEaTe DaTaBaSe";

char *minusculas = toupper (cadena); // retorna "CREATE DATABASE"
