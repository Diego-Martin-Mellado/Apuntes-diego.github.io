Indice - C++.

<ctypes.h >



Es la librería para clasificar y transformar caracteres de manera individual.

<math.h >



Conjunto de funciones que sirven para simplificar el cálculo de las operaciones matemáticas más comunes.

<ncurses.h >



Librería de gráficos por consola exclusiva de sistemas Linux.

<stdint.h >



Librería para facilitar el uso de datos numéricos.

<stdio.h >



Standard Input / Output. Librería para manejar flujo de datos por consola.

<stdlib.h >



Standard Library. Funciones de propósito general incluyendo almacenamiento dinámico en memoria, aleatoriedad, ordenamiento, entorno.

<string.h >



Librería con funciones para el manejo de cadenas de caracteres de manera más sencilla de la normal.

<time.h >



Conjunto de funciones para manipular información acerca de fechas y tiempos.

<unistd.h >



En los lenguajes de programación C y C++, unistd.h es el nombre del archivo de encabezado que proporciona acceso a la API del sistema operativo POSIX.

<windows.h >



La libreria de Windows de C++, poco recomendable de usar, pero ahí está.
