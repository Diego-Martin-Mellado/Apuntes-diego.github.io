Función usleep.

usleep () es una función que está en la librería <unistd.h>

usleep () hace una pausa en el programa del tiempo indicado en millonésimas de segundos, después, continúa con las siguientes instrucciones.

Sintaxis:

usleep ( int microsegundos );
Referencia:

  • Microsegundos:
    Es el número de millonésimas de segundo que el programa va a estar esperando antes de ejecutar la siguiente instrucción.

Retorno:

void

Ejemplos de uso de usleep.

Ejemplos de uso de usleep ():

usleep (1000000); // Espera 1 segundo

usleep (3456789); // Espera 3.456789 segundos
