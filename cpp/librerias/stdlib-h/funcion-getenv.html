Función getenv.

getenv () es una función que se encuentra en la librería <stdlib.h>.

getenv () sirve para obtener el valor de una variable de entorno del sistema operativo del ordenador del cliente.

Devuelve un puntero a una cadena de caracteres. (char *)

Sintaxis:

getenv ( char * variable);

Referencia:

  • Variable:
    Variable de entorno de la cual se va a obtener información.

Retorno de getenv ().

char *

Ejemplo de uso de getenv.

Uso de getenv ():

char *path = getenv ("PATH") // Recoje la información que contiene la variable PATH del sistema

char *user = getenv ("USERNAME") // Recoje la información que contiene la variable USERNAME del sistema
