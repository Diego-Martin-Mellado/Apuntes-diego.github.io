Función system.

system () es una función que se encuentra en la librería <stdlib.h>.

system () se utiliza cuando quieres ejecutar un comando de la consola del cliente.

Conviene saber en qué sistema operativo se está ejecutando tu programa antes de ejecutar un comando con system ().

Sintaxis:

system ( const char * comando );

Referencia:

  • Comando:
    Es un string que contiene un comando a ejecutar en la consola del cliente.

El parámetro de system () no puede contener partes variables formateadas.
Retorno:

El que retorne el comando ejecutado.

Ejemplos de uso de system.

Ejemplos de uso de system ():

// Limpia la consola de Linux
system ("clear");

// Limpia la consola de Windows
system ("cls");
