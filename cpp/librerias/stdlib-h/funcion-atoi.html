Función atoi.

La función atoi () es una función que está en la librería <stdlib.h>.

La función atoi () (ASCII to integer) convierte caracteres ASCII a números enteros.

Para que la función atoi () funcione correctamente, el parámetro debe contener solamente números.

Sintaxis:

atoi (char * cadena);

Referencia:

  • Cadena:
    Es el string que se va a pasar a números enteros.

Retorno:

int

Ejemplos de uso de atoi.

Ejemplos de uso de la función atoi ().

#include <stdio.h> // Encabezados
#include <stdlib.h>

// Funcion main
int main (int argc, char *argv[]) {

  // Código a ejecutar
  char *pal = "73";
  int num = atoi (pal); // "num" vale 73

  return EXIT_SUCCESS;
}
