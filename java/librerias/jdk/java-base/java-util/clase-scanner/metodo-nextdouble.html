Método nextDouble - Java.

nextDouble () es un método que está en la clase Scanner que se encuentra en la librería java.util de Java.

nextDouble () indica al objeto de clase Scanner que lo siguiente que introduzca el usuario será convertido a datos de tipo double.

Si el usuario introduce un dato que no es double, nextDouble () resultará un error que finalizará el programa.
Sintaxis:

Scanner.nextDouble ();
Retorno:

double

Ejemplos de uso de nextDouble

Uso del método Scanner.nextDouble ():

public class Programa {
  public static void main (String [] args) {
    double numero;
    Scanner teclado = new Scanner(System.in); // Creo el teclado
    System.out.println ("Introduce un double: ");
    numero = teclado.nextDouble(); // Recoje un double
    teclado.close();
  }
}
