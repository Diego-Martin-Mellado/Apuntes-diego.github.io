Método nextBoolean - Java.

nextBoolean () es un método que está en la clase Scanner que se encuentra en la librería java.util de Java.

nextBoolean () indica al objeto de clase Scanner que lo siguiente que introduzca el usuario será convertido a datos de tipo boolean.

Si el usuario introduce un dato que no es boolean, nextBoolean () resultará un error que finalizará el programa.
Sintaxis:

Scanner.nextBoolean ();
Retorno:

double
Excepciones que puede lanzar:

  • InputMisMatchException Si el siguiente valor no es un valor booleano válido.
  • NoSuchElementException Si no hay un valor que se pueda parsear.
  • IllegalStateException Si el objeto Scanner ha sido cerrado con el método close ().

Ejemplos de uso de nextBoolean

Uso del método Scanner.nextBoolean ():

public class Programa {
  public static void main (String [] args) {
    boolean condicion;
    Scanner teclado = new Scanner (System.in); // Creo el teclado
    System.out.println ("Introduce un booleano válido: ");
    condicion = teclado.nextBoolean (); // Recoje un double
    // Random stuff con el booleano
    teclado.close ();
  }
}
