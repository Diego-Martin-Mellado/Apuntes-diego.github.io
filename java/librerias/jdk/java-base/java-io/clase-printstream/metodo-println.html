Método println - Java.

println () es un método que está en la clase System.out.

println () imprime el texto indicado en su parámetro por el canal de salida de datos definido por la clase System.out.

println () tiene un solo parámetro que es el String que se va a imprimir por pantalla.

Si el parámetro de println () no es un String, println () lo convertirá a String antes de imprimirlo por consola.
En cada línea de código escrita de ésta función, habrá un salto de línea a continuación, debido a que println () lo pone automáticamente. Ésta es la diferencia con print ().

Sintaxis:

System.out.println ( String dato );
System.out.println ( boolean dato );
System.out.println ( char dato );
System.out.println ( char[] dato );
System.out.println ( double dato );
System.out.println ( float dato );
System.out.println ( int dato );
System.out.println ( long dato );
System.out.println ( Object dato );

Referencia:

  • Dato:
    Es el dato o valor de variable que se va a imprimir por consola, como se puede ver, puede ser de cualquier tipo.

Retorno:

void

Ejemplos de uso de println

Uso del método System.out.println ():

public class Programa {
  public static void main (String [] args) {
    System.out.println ("Hola mundo"); // Imprime "Hola mundo"
    System.out.println (true); // Imprime true
    System.out.println (73); // Imprime 73
  }
}
