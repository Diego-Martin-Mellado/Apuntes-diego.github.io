Método close - Java.

close() es un método que está en la interfaz Closeable que se encuentra en la librería java.util.

close() finaliza el inputStream creado por la instanciación de una clase Scanner.

Sintaxis:

Scanner.close();
Retorno:

void

Ejemplos de uso de close

Uso del método Scanner.close ():

import java.util.*;

public class Programa {
  public static void main (String [] args) {
    Scanner teclado = new Scanner(System.in);
    // Some stuff
    teclado.close();
  }
}
