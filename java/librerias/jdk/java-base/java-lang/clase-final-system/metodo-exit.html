Método exit - Java.

exit () es un método que se encuentra en la librería java.lang que es la librería estándar de Java.

getProperty () es un método que se encuentra en la clase System.

Éste método indica que el programa debe finalizar, y su parámtero es un número que sólo puede ser 0 en caso exitoso, 1 - 127 errores definidos por el programador y 128 - 255 son los definidos por el sistema.

Sintaxis:

System.exit ( int Éxito );

Referencia:

  • Éxito:
    Indica si la ejecución del programa se ha realizado con éxito o no al llegar a ese punto.

Retorno:

void

Ejemplos de uso de exit

Uso del método System.exit ():

public class Programa {
  public static void main (String [] args) {
    System.exit ( 0 ); // Termina el programa con éxito
    System.out.println ("Esto no se ejecuta"); // Ésta línea no se ejecuta porque el programa ha acabado en la línea de arriba
  }
}
