Clase final System - Java.

La clase System representa al sistema donde se esta ejecutando el programa Java. Por lo tanto se la puede utilizar para interactuar con el entorno en el que corre, y utilizar las propiedades del entorno, sistema operativo, usuario y demás.

La clase System es final, por lo que no es una clase instanciable, es decir, no debes crear objetos con ésta clase.

Clases y métodos de System

Método exit



Termina la máquina virtual Java que se está ejecutando actualmente.

Método gc



Ejecuta el recolector de basura.

Método getProperty



Obtiene la propiedad del sistema indicada por la clave especificada.
