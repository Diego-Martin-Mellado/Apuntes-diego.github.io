Método gc - Java.

gc () es un método que se encuentra en la librería java.lang que es la librería estándar de Java.

gc () es un método que se encuentra en la clase System.

gc () ejecuta el Garbage Collector del sistema. El garbage collector recicla los objetos de la memoria de la máquina virtual de Java (JVM) para optimizar su uso y su espacio.

gc () No lanza excepciones.

Sintaxis:

System.gc ();
Retorno:

void
