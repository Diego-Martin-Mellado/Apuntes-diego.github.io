java.lang - Java.

Es un paquete de Java que proporciona clases fundamentales en el diseño de la sintaxis básica de Java, pdría decirse que es donde se "aloja" el lenguaje Java.

De hecho, al ser el paqueta básico, no necesita ser importado como paquete.

Clases y funciones de java.lang - Java.

Clase final System



Contiene el marco de recopilación, las clases de recopilación heredadas, el modelo de evento, las instalaciones de fecha y hora, la internacionalización y las clases de utilidad misceláneas (un tokenizador de cadenas, un generador de números aleatorios y una matriz de bits).

Clase String



Representa el tipo de dato 'Cadena de caracteres' en Java.
