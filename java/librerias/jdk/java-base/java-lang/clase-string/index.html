Clase String - Java.

La clase String representa el tipo de dato 'String' o 'Cadena de caracteres' en Java, al cual pertenecen una serie de métodos públicos que facilitan su uso al programador.

Puedes crear las variables de tipo String que necesites sin pasar la memoria de la JVM.

Objetos padre:

  • java.lang.Object

Interfaces implementadas:

  • Serializable
  • Comparable <string>
  • CharSequence
