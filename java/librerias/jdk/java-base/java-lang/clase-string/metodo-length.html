Método String.length - Java.

El método String.length() retorna o devuelve el número de caracteres que tiene el String al que se aplica el método.

Sintaxis:

String.length()
Retorno:

int

Ejemplos de length

Se puede usar para iterar en un String, o para actuar de una manera u otra según la longitud de un String.

String nombre = "Sandra";
for (int i = 0; i < nombre.length(); i++)
  System.out.println(nombre.charAt(i));
