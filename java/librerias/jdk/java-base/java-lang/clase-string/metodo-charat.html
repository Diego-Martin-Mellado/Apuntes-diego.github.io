Método String.charAt - Java.

El método String.charAt() retorna o devuelve el caracter de un String que se encuentra en la posición indicada por parámetro.

Sintaxis:

String.charAt(int posicion)
Referencia:
  • Posicion:
    Es la posición de la cual se quiere extraer el caracter.
Retorno:

char

Ejemplos de charAt

Se puede usar para iterar en un String, o para actuar de una manera u otra según la longitud de un String.

String nombre = "Victor";
System.out.println(nombre.charAt(0)); // Devuelve una letra 'V'
